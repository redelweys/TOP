Должна быть возможность добавить задачу в расписание:- Через интерфейс: кнопка “Добавить в расписание“- SOAP запрос AddTaskInCron (в ответе будет, что задача добавлена в расписание + дата следующего запуска. Например, «Расписание успешно добавлено в задачу 18 Следующая дата запуска 06.04.2017 12:00»)- REST запрос AddTaskInCron (в ответе будет, что задача добавлена в расписание + дата следующего запуска. Например, «Расписание успешно добавлено в задачу 18 Следующая дата запуска 06.04.2017 12:00»)

Должна быть возможность изменить задачу:- Через интерфейс: кнопка “Изменить”- SOAP запрос UpdateTask (в ответе будет «Задача успешно изменена!»)- REST запрос UpdateTask (в ответе будет «Задача успешно изменена!»)

Должна быть возможность удалить задачу:- Через интерфейс: кнопка “Удалить“- SOAP запрос DeleteTask (вернет текст «Задача с ID 17 успешно удалена». Чтобы удалить задачу, нужно знать ее ID)- REST запрос DeleteTask (вернет текст «Задача с ID 17 успешно удалена». Чтобы удалить задачу, нужно знать ее ID)

В режиме просмотра / редактирования задачи в интерфейсе должно быть видно ID задачи

ID должно возвращаться в ответе метода CreateTask

Обычный пользователь должен видеть только свои задачи, менеджер - все существующие в базе

Обычный пользователь может изменять только свои задачи, менеджер - все существующие в базе

Обычный пользователь может удалять только свои задачи, менеджер - все существующие в базе
